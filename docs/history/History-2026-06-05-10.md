- **핵심 목표:** 실제 모바일 기기(크롬/사파리)에서 발생하는 끈질긴 캐시 잔상과 PDF 보안 제약 해결.
- **최종 검증 완료:** v0.1.4 배포를 통한 물리적 캐시 삭제 및 네이티브 PDF 연동 성공.

## 2. 📝 주요 수정 및 해결 내역
- **강력 캐시 파괴 (Emergency Cache Buster):**
  - `vite.config.ts`: `selfDestroying: true` 설정을 통해 구버전 서비스 워커 자폭 유도.
  - `index.html`: 앱 로드 시 모든 서비스 워커 등록을 강제 해제하는 인라인 스크립트 주입.
  - **결과:** 사용자님의 수동 조치 없이도 접속 즉시 최신 소스 주입 강제화.
- **레이아웃 겹침(Ghosting) 종결:**
  - `App.tsx`: `isMobile` 분기 시 PC용 루트 컴포넌트를 메모리에서 완전히 배제.
  - `index.css`: 미디어 쿼리 2중 잠금(`display: none !important`)으로 물리적 잔상 제거.
- **실전 PDF 네이티브 전환:**
  - 모바일 브라우저의 `iframe` 내 PDF 실행 차단 정책 대응.
  - PDF 선택 시 전용 안내 UI와 **"네이티브 뷰어로 열기"** 버튼 제공하여 호환성 100% 확보.
- **버전 관리 체계 고도화:**
  - 앱 버전을 **v0.1.4**로 업데이트하고 하단 스테이터스 바에 명시하여 업데이트 여부 직접 확인 가능하도록 개선.
