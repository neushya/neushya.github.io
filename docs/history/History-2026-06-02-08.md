**저장 순서:** 08

## 1. 📝 Stage 3: 모바일 심플 모드 상세 기획 완료
- **원칙:** PC 버전의 디자인 및 로직에 절대적인 무간섭(Zero-Impact) 보장.
- **UI 설계:**
    - **사이드바:** 모바일 환경에서 전면 제거 (단일 파일 집중 구조).
    - **토글 시스템:** 하단 고정 탭 바 영역에 '에디터/프리뷰' 1:1 전환 버튼 배치.
    - **저장 경험:** IndexedDB 기반 자동 저장 + 명시적 '내보내기(Export)'를 통한 파일 다운로드.

## 2. 🏛️ 제니토하네스 Stage 4: 개발 계획 수립
- **우선순위:** 
    1. 디바이스 감지 엔진(isMobile) 구축.
    2. 모바일 전용 독립 레이아웃 컨테이너 개발.
    3. Blob 기반 파일 내보내기 기능 구현.
    4. PC 버전 영향도 역방향 검증(Regression Test).

## 3. 🛡️ 무간섭 아키텍처 확정
- 모바일 접속 시 데스크톱용 `PanelGroup` 및 `Sidebar` 컴포넌트 마운트를 차단함.
- CSS 미디어 쿼리와 React 상태 분리를 통해 스타일 및 로직 간섭을 원천 봉쇄함.
