## 2. 📝 주요 수정 및 결함 해결 내역
- **TypeScript 빌드 오류 수정:** `App.tsx` 내 미사용 임포트(`del`) 제거를 통해 GitHub Actions 빌드 중단 결함 해결.
- **파일 열기 범용성 강화:** MIME 타입 필터링을 세분화하여 `.js`, `.ts`, `.json`, `.sql` 등 모든 개발 문서를 제약 없이 열 수 있도록 개선.
- **미디어 파일 차단:** 이미지, 영상, 압축 파일 등 비문서 파일 선택 시 "문서 파일이 아닙니다." 경고 팝업 로직 구현.
- **PDF 전용 뷰어 통합:** PDF 파일을 텍스트가 아닌 실제 문서 레이아웃으로 볼 수 있는 브라우저 내장 뷰어(iframe 기반) 엔진 탑재.
- **백색 화면(White Screen) 해결:** 구식 `gh-pages` 브랜치 방식에서 최신 **GitHub Actions 직접 배포(Direct Deployment)** 방식으로 전환하여 경로 및 동기화 오류 완벽 차단.

## 3. ✨ UX 및 기능 고도화
- **하이브리드 자동 저장:** 기존 파일은 로컬 쓰기, 신규 파일은 IndexedDB 초안(Draft) 저장을 통해 데이터 유실 방지.
- **PWA 최적화:** 설치형 웹 앱 환경 구축 및 오프라인 모드 지원.
- **최단 도메인 적용:** 저장소 이름 변경을 통해 `neushya.github.io` 루트 도메인 즉시 배포 성공.
